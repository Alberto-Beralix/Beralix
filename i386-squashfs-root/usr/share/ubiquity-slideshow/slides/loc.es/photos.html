Diviértase con sus fotos

Shotwell es un práctico administrador de imágenes que está listo para sus dispositivos. Conecte una cámara o un teléfono para transferir sus fotos, es muy fácil compartirlas y mantenerlas seguras. Si se siente creativo, puede probar muchas aplicaciones para fotografía en el Centro de software de Ubuntu.

Software incluido

  • Gestor de fotos Shotwell

Software compatible

  • Editor de imágenes GIMP

  • Editor de vídeo Pitivi
